### C 语言

>任何比 C 语言更低级的语言，都不足以完整地抽象一个计算机系统；任何比 C 高级的语言，都可以用 C 来实现。

C 语言提供了对底层细节的强大操控能力。绕过 C 语言完成对计算机底层细节的理解是困难的。

C 语言的内存管理或者说指针的的使用是 C 语言中的重点。

### 书籍推荐

这两本书都对 C11 标准的绝大多数细节做了完整的讲解。`C 语言编程魔法书：基于 C11 标准` 还讲解了 `gcc` 与 `clang` 编译器实现的扩展。

- `C Primer Plus`
- `C 语言编程魔法书：基于 C11 标准`

对于指针与内存管理，也可参考：
- `C 和指针`

# 语言学习

### 为什么现在就推荐大一的你学习一下新的语言：

你可以看到更广阔的编程世界。

### 为什么你可以不选择学新的语言

想要之后做 Kernel, Server，DB 的同学，把 C 语言基础打好比学花丽胡哨的新语言更重要。
- 段错误遇到过吧？怎么解决呢？
- 栈溢出遇到过吧？怎么解决呢？
- 内存泄漏遇到过吧？怎么解决呢？
- 基础数据结构算法实现过没？不想成为调包侠吧？
